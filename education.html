Other Notable Courses

Course Description
ITM780 - Web Design and Management
This course covers the fundamentals of developing and managing business websites using HTML5, CSS, and JavaScript. It emphasizes best practices, web standards, and practical skills for publishing and managing sites. Topics include backup, disaster recovery, and maintaining a professional web presence..
ITM200 - Fundamentals of Programming
This course covers object-oriented and event-driven programming principles in a business context, emphasizing logic development, modular design, testing, and debugging. Concepts like variables, loops, arrays, and error handling are taught using Python in practical lab sessions.
ITM500 - Database Management and Design
This course introduces key concepts of data and information management, including data modeling and relational databases. It covers database administration, data quality, and security essentials. Structured Query Language (SQL) is explored for decision support applications.
